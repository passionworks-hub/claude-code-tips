# Claude Code Tips

Practical, copy-paste-ready tricks for getting more out of [Claude Code](https://claude.com/claude-code) — Anthropic's agentic coding CLI.

Each tip is small, self-contained, and costs **zero tokens** unless noted otherwise.

## Tips

| # | Tip | What it does |
|---|---|---|
| 1 | [🔊 Voice alert when input is needed](tips/voice-alert.md) | Your Mac *says your name* out loud when Claude Code is waiting on you — no more missed prompts. Zero tokens, works offline. *(macOS)* |
